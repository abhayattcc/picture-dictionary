## 🛠 How to Use

1. Visit **[Best Odia Dictionary](https://abhayattcc.github.io/Best-odia-dictionary/)**
2. Type any Odia, English or Hindi word
3. Click **🔊 Speak** to hear pronunciation
4. Click **✏️ Edit** to modify or save the word
5. Use **Export** button to download your word list

**Pro Tip**: Works perfectly on mobile – great for school students in Odisha!
